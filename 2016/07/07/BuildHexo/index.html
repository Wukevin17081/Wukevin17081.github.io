前言

其實在這之前我有試過用 octepress, 但是那個要先裝 ruby, 偏偏 ruby 對 windows 的支援性好像不是很好，裝了很久環境就是建不起來。
最後找到用 hexo 也可以建，而且網路的教學都還滿簡單，所以就決定是你了，本來想說應該半天就可以弄好了，但是不知道怎麼摸魚的，還是花了一天 ….

到目前為止是電腦端的設定，再來要建立 github 了。

Github

先到 Github 上申請一個帳號，然後建一個 repository, 這個 repository 的名稱必須跟你的帳號一致。必如你的帳號是 testacc, 那你的 Res 就要叫 testacc.github.io,

建立完帳戶後，你還要有 ssh 的 key, 不然你部署的時候會出錯。
詳細的說明請看 github 的 help, 我這裡簡述一下。
